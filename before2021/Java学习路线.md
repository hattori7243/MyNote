# <center> Java </center>

## Java技术体系

1. Java程序设计语言
2. 各种平台上的Java虚拟机（JVM）
3. Java API类库
4. 一系列辅助工具，如JavaC

#### 1+2+3+4=JDK

#### 2+3=JRE（Java运行时环境）

### Java技术体系所划分的三大平台

#### JavaSE vs JavaEE vs JavaME

- EE企业版，主要用于企业开发

#### 推荐 JDK1.8
